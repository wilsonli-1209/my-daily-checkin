# Daywell 云端网站

1. 在 Supabase 创建免费项目，在 SQL Editor 运行 `supabase-schema.sql`。
2. 将 Settings → API 中的 Project URL 和 anon key 填入 `config.js`。
3. 把 `index.html`、`style.css`、`app.js`、`config.js` 上传至 Netlify、Vercel 或 GitHub Pages。
4. 在 Supabase Authentication → URL Configuration 中加入网站网址。

同一邮箱账号在任何设备登录后，打卡、备注和图片都会同步。用户只能访问自己的数据。
